# Docker Basics

## Comandos útiles
1. Listar los contenedores corriendo:
``` bash
docker ps
```

2. Denere un contenedor usando su ID:
```bash
docker stop <CONTAINER ID>
```

3. Construir una imágen:
```bash
docker build -t <TAG NAME>
```

3. Crear un volúmen:
```bash
docker volume create <VOLUME NAME>
```
